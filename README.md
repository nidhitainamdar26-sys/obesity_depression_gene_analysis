# Obesity–Depression Gene Expression Analysis

## Project Overview
This project performs a preliminary gene expression analysis to study the comorbidity between obesity and depression using Python.

## Methods
- Data handling using Pandas
- Statistical analysis (descriptive statistics, grouping)
- Data visualization using Matplotlib

## Biological Relevance
The selected genes (TNF, IL6, LEP, BDNF, INS, CRP) are associated with inflammation, metabolic regulation, and neurobiology, which are key mechanisms linking obesity and depression.

## Results
The analysis indicates higher gene expression levels in obesity–depression comorbidity compared to individual conditions.

## Note
This dataset is simulated and used for demonstrating computational analysis workflow.
